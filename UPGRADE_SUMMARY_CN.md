### 2. 新增特征类别：信用与流动性 (28个新特征)

#### A. 信用利差特征 (HYG/LQD) - 6个特征

```python
# 核心指标
credit_spread_ratio = HYG / LQD  # 比率下降 = 信用利差扩大 = 风险上升

# 关键特征
- credit_spread_change_5d    # 5日变化
- credit_spread_change_20d   # 20日变化
- credit_stress              # 利差扩大警报
- credit_spread_percentile   # 历史分位数
- credit_spread_vs_ma        # 相对均线位置
```

**特征重要性排名:** #14 (1.66%)
**解释:** 信用利差扩大是系统性风险的领先指标

#### B. SKEW指数特征 (尾部风险) - 6个特征

```python
# 核心指标
skew_level  # SKEW值 (>135 = 尾部风险上升, >145 = 极端风险)

# 关键特征

- skew_change_20d     # 20日变化

```

**解释:** SKEW指数衡量"黑天鹅"事件概率

#### C. 黄金+VIX同涨信号 - 4个特征

```python
# 核心信号
```
